Quick StartInicio Rápido

Welcome to the olympiad calculatorBienvenido a la calculadora olímpica

  1. 1 Open the side menu (☰) and select Special Functions for the classic calculator keypad (~40 functions). Abre el menú lateral (☰) y selecciona Funciones Especiales para el teclado clásico (~40 funciones).
  2. 2 Enter a number and press any function button. Ingresa un número y presiona cualquier botón de función.
  3. 3 If the function needs more values, a pending operation indicator appears on screen. Si la función necesita más valores, aparece un indicador de operación pendiente en pantalla.
  4. 4 For exact olympiad tools (fractions, geometry, polynomials, …) open the menu → Olympiad Tools. Para herramientas olímpicas exactas (fracciones, geometría, polinomios, …) abre el menú → Herramientas de Olimpiada.
  5. 5 The side panel (tablet) or bottom panel (mobile) shows automatic analysis of every entered number. El panel lateral (tablet) o inferior (móvil) muestra el análisis automático de cada número ingresado.
1-parameter functions execute immediately. 2+ parameter functions show an indicator and wait for more values. Las funciones de 1 parámetro se ejecutan de inmediato. Las de 2+ parámetros muestran un indicador y esperan más valores.

Counts how many integers from 1 to n are coprime with n (GCD(k,n)=1).Cuenta cuántos enteros del 1 al n son coprimos con n (MCD(k,n)=1).

  • Σ φ(d) for d|n = n

For positive numbers it always holds that:

Equality holds only when all values are equal.
